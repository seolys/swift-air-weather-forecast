# swift-air-weather-forecast

swift로 구현한 iOS앱 날씨먼지입니다.  
언어를 학습하면서 오픈API와 오픈이미지를 활용하여 구현하였습니다. 

앱스토어에 등록하지않은 어플리케이션 입니다.

소스에 사용된 공공데이터포털 API는 2020/4/28 행안부에서 정비/통합으로 인해 폐기예정이며  
변경된API로 이 필요합니다.
